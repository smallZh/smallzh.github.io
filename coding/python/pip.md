# 使用pip

## 1 升级pip
```shell
python -m pip install --upgrade pip
```
## 2 pip使用
### 2.1 安装包
```shell
pip install [包名]
```
### 2.2 搜索包
```shell
pip search [包名]
```
### 2.3 查看包信息
```shell
pip show --files [包名]
```
### 2.4 查看有更新的包
```shell
pip list --outdated
```
### 2.5 更新包
```shell
pip install --upgrade [包名]
```
### 2.6 卸载包
```shell
pip uninstall [包名]
```
